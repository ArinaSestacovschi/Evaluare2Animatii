# Joc Canvas Animat - Evaluare

Acest mic proiect demonstrează utilizarea `canvas`, obiecte JavaScript (player, enemy, proiectile), controlul ratei cadrelor cu `setTimeout`, import de grafică SVG, filtre CSS și transformări.

Instrucțiuni rapide:
- Deschide `index.html` în browser.
- Mută mouse-ul pentru a deplasa nava și click pentru a trage.
- Ajustează sliderul `Setare FPS` pentru a testa rata cadrelor (folosește `setTimeout`).
